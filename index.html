Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language. It is used to create the structure of a webpage.
  • It is made up of elements.
  • The 'head' element contains information about the webpage.
  • The 'body' element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GitHub

  • Repository "repo": every project on Github is called a repo. It's a folder that contains all of the files & revisions for your project
  • Branch: workspace where you can make changes that don't affect the live site
  • Markdown: (.md) is a file format
  • Commit: saving a new version of a file you're working on
  • Pull Request: PR asking for your changes to merge into another branch
  • Merge: When a PR is approved and pulled into the main branch

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
